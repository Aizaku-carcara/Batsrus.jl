SWMF.convertVTK — Function
convertVTK(head, data, connectivity, filename)

Convert 3D unstructured Tecplot data to VTK. Note that if using voxel type data in VTK, the connectivity sequence is different from Tecplot.

SWMF.readdata — Method
readdata(filenameIn, (, dir=".", npict=1, verbose=false))

Read data from BATSRUS output files. Stores the npict snapshot from an ascii or binary data file into the coordinates x and data w arrays. Filenames can be provided with wildcards.

Examples

filename = "1d_raw*"
data = readdata(filename)

SWMF.readtecdata — Method
readtecdata(filename, IsBinary=false, verbose=false)

Return header, data and connectivity from BATSRUS Tecplot outputs. Both binary and ASCII formats are supported. The default is reading pure ASCII data.

Examples

filename = "3d_ascii.dat"
head, data, connectivity = readtecdata(filename)
